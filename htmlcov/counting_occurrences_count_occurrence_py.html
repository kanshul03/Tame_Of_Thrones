7# This class is used to update the character occurrence count list(occurrence_count). 

8# occurrence_count is basically a list of length 26, denoting all alphabets with a default value of 0, which means the 

9# character hasn't occurred anywhere yet. If a character occurs in animal name, we give it's value an increment. 

10# If it occurs in decrypted message, then we give it's value a decrement. In the end, if all the values are zero or 

11# negative in character occurrence count list, then it means that all the characters in the animal name has occurred 

12# in the decrypted message as well. 

13class CountingOccurrence(implements(CountingOccurrenceInterface)):
